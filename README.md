# craw4u

crawler for you

## Usage

- `Request`<br>
Example:
```golang
craw4u.Request("GET", "https://github.com/YanHerChen/craw4u")
```
- `Get`<br>
Example:
```golang
craw4u.Get("https://github.com/YanHerChen/craw4u")
```
